# 侧边栏

## 1. 模块定位

侧边栏是“当前网页即时处理”的主工作台，负责页面提取、快速提问、快捷输入、分支分析和导出。

## 2. 核心抽象

- 当前页面上下文
- 提取内容区
- 快捷输入标签区
- 聊天区
- 分支区
- 输入区

## 3. 能力边界

负责：

- 展示当前页面信息和提取内容。
- 展示并切换标签。
- 发起发送、停止、清空、导出。
- 展示主回答与分支回答。
- 发起消息编辑、消息重试、分支继续新增、分支停止和分支删除。
- 呈现黑名单确认层和提取错误态。

不负责：

- 直接访问远端模型或同步后端。
- 直接维护持久化 key。
- 直接决定 MV3 权限和脚本注入策略。

## 4. 对外接口

- one-shot command：
  - 获取页面信息
  - 确认黑名单放行
  - 切换提取方式
  - 重新提取
  - 发送消息
  - 停止会话
  - 停止分支
  - 删除分支
  - 编辑用户消息
  - 重试消息
  - 继续新增分支
  - 清空当前标签
  - 导出对话
- long-lived port：
  - 订阅流式 chunk、完成、错误、loading 更新

## 5. 关键流程

- 打开侧边栏并优先展示缓存。
- 内容提取与方法切换。
- 文本/图片发送。
- 自动触发快捷输入。
- 分支并发与预览。
- 消息编辑、重试和分支局部操作。
- 侧边栏关闭重开后的 loading 恢复。

## 6. 错误与异常处理

- content script 未连接时，先自动刷新当前页面一次并重试，仍失败后再提示用户手动重试。
- 提取失败时保留旧内容和当前页面上下文。
- 流式错误时回收 loading 并标记目标消息或分支。
- 用户取消不显示系统错误。
- 黑名单命中时先阻断自动流程。

## 7. 数据与状态

- UI 运行态：
  - 当前活动标签
  - 输入框内容
  - 本地预览图片
  - 弹层显示状态
- 页面级持久化辅助状态：
  - 标签初始化状态
  - 自动触发状态
- 持久化依赖：
  - `PageRecord`
  - `ConversationRecord`
  - `LoadingStateRecord`
  - `ExtensionConfig`

## 9. 约束与禁止事项

- 不能在 UI 直接使用 provider SDK。
- 不能假设 side panel 始终处于打开状态。
- 不能通过 DOM 状态反推持久化事实。
- 不能把自动触发设计成强制切换当前用户视图。
- 清空当前标签时必须同步重置该标签的自动触发初始化状态。

## 10. 测试要求

- 职责测试：打开侧边栏、提取、发送、分支、导出。
- 边界测试：受限页退化、黑名单拦截、配置不完整模型禁止发送。
- 正常流测试：文本、图片、快捷输入、自动触发。
- 错误流测试：提取失败、流式错误、content script 未连接。
- 异常流测试：side panel 重开恢复、取消请求、分支局部删除、编辑消息后重发。
- 不变量测试：页面级 `includePageContent` 恢复、标签隔离。
- 可观测性测试：关键 loading 和错误事件可见。
