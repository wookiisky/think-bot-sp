# 对话管理页

## 1. 模块定位

对话管理页是历史页面和历史会话的工作台，不是归档查看器。它负责按页面维度恢复上下文并继续工作。

## 2. 界面布局映射

对话管理页界面按设计图拆成 3 个稳定区域：

- 左侧历史栏：
  - 固定包含搜索框、历史页面列表和空状态。
  - 支持高密度列表浏览，列表项按最近更新时间排序。
  - 单条页面项固定包含网站图标、标题、打开原网页按钮和删除按钮。
- 中间分隔条：
  - 支持左右栏宽度拖拽。
  - 只负责布局尺寸调整，不承载业务动作。
- 右侧工作区：
  - 顶部是页面标题和 URL 信息区，标题支持内联编辑。
  - 标题区下方是提取内容区，承担历史正文阅读职责。
  - 提取内容区下方是快捷输入标签区和聊天区。
  - 底部是继续对话输入区，包含图片预览、模型选择和发送动作。

## 3. 核心抽象

- 历史页面列表
- 选中页面上下文
- 提取内容区
- 恢复后的标签和聊天区
- 图片预览区
- 继续对话输入区

## 4. 能力边界

负责：

- 加载历史页面列表。
- 搜索、切换、删除页面。
- 恢复页面标题、URL、提取内容、标签和聊天。
- 编辑页面标题。
- 打开原网页、复制提取内容。
- 展示并切换快捷输入标签。
- 展示主回答、分支回答和 Markdown 内容。
- 在历史上下文上继续对话，支持图片输入、模型切换、停止、清空和导出。
- 恢复页面级和标签级 loading 状态。

不负责：

- 重新定义聊天协议。
- 直接修改同步远端。
- 直接执行页面 DOM 提取。

## 5. 对外接口

- 获取页面列表。
- 搜索页面。
- 获取单页面全部恢复数据。
- 编辑标题。
- 删除页面。
- 发送消息、停止会话、清空当前标签、编辑消息、重试消息、继续新增分支、停止分支、删除分支、导出。
- 订阅流式 chunk、完成、错误和 loading 更新。

## 6. 关键流程

- 首次加载历史页面列表。
- 搜索过滤。
- 选中页面恢复完整上下文。
- 拖拽调整左右栏宽度。
- 标题编辑并同步左侧列表。
- 复制提取内容和在新标签页打开原网页。
- 历史会话上继续提问，支持图片输入和模型切换。
- 停止当前标签会话、清空当前标签会话、导出当前标签会话。
- 恢复主会话与分支 loading。
- 删除页面并按同步状态选择软删或硬删。

## 7. 错误与异常处理

- 页面记录损坏时只影响当前右侧，不影响其他页面浏览。
- 标题保存失败时回退旧标题。
- 删除失败时不提前清空右侧视图。
- loading 恢复失败时降级为静态历史视图，不阻断其他页面切换。

## 8. 数据与状态

- UI 运行态：
  - 当前搜索词
  - 当前选中页面
  - 左右栏宽度
  - 列表滚动位置
  - 标题编辑态
  - 当前活动标签
  - 右侧页面恢复状态
  - 提取内容区滚动位置
  - 聊天区滚动位置
  - 输入框草稿
  - 当前选中模型
  - 本地图片预览
  - 分支预览层显示状态
- 持久化依赖：
  - `PageRecord`
  - `ConversationRecord`
  - `LoadingStateRecord`
  - `ExtensionConfig`

页面级动作与标签级动作必须严格分离：

- 页面级动作：
  - 选中页面
  - 编辑标题
  - 复制提取内容
  - 打开原网页
  - 删除页面
- 标签级动作：
  - 切换标签
  - 发送消息
  - 停止当前标签会话
  - 清空当前标签会话
  - 导出当前标签会话
  - 编辑消息
  - 管理分支

## 10. 约束与禁止事项

- 左侧列表必须以页面元数据为 source of truth。
- 左侧列表项固定展示图标、标题、打开按钮和删除按钮，不能把关键动作折叠到二级入口。
- 删除按钮不能顺带触发页面选中。
- 打开原网页只开新标签，不改变右侧当前选中页面。
- 右侧必须同时呈现提取内容区和聊天区，不能把提取内容退化为某个标签的临时消息。
- 快捷输入标签必须保持高密度、多行换行布局，不能要求横向滚动访问常用标签。
- 右侧输入区行为要与侧边栏保持一致，包括模型选择、图片输入、停止、清空、导出和分支交互。
- 左右栏拖拽只影响布局宽度，不改变当前页面、标签、输入草稿和滚动上下文。
- 继续对话行为要与侧边栏保持一致。

## 11. 测试要求

- 职责测试：列表、搜索、恢复、继续对话、标题编辑、删除。
- 职责测试：分隔条拖拽、标签切换、图片预览、模型切换、停止、清空、导出。
- 正常流测试：恢复已有页面并继续提问、编辑消息、继续新增分支。
- 正常流测试：复制提取内容、打开原网页不改变当前选中。
- 正常流测试：标题 `Enter` 保存、`Esc` 取消、失焦保存。
- 边界测试：空状态、无搜索结果、损坏页面数据、当前选中页面被删除。
- 错误流测试：标题保存失败、删除失败。
- 异常流测试：有 loading 状态时恢复页面，主会话和分支状态都可见。
- 删除语义测试：同步开启时软删、未开启时硬删。
- 不变量测试：删除页面不影响其他页面记录。
- 不变量测试：左右栏拖拽和打开原网页不影响当前右侧上下文。
- 可观测性测试：列表更新时间、loading 状态和错误状态清晰可见。
