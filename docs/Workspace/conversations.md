# 对话管理页

## 1. 模块定位

对话管理页是历史页面和历史会话的工作台，不是归档查看器。它负责按页面维度恢复上下文并继续工作。

## 2. 核心抽象

- 历史页面列表
- 选中页面上下文
- 恢复后的标签和聊天区
- 继续对话输入区

## 3. 能力边界

负责：

- 加载历史页面列表。
- 搜索、切换、删除页面。
- 恢复页面内容、标签和聊天。
- 编辑页面标题。
- 打开原网页、复制提取内容。
- 在历史上下文上继续对话。

不负责：

- 重新定义聊天协议。
- 直接修改同步远端。
- 直接执行页面 DOM 提取。

## 4. 对外接口

- 获取页面列表。
- 搜索页面。
- 获取单页面全部恢复数据。
- 编辑标题。
- 删除页面。
- 发送消息、停止会话、编辑消息、重试消息、继续新增分支、停止分支、删除分支、导出。

## 5. 关键流程

- 首次加载历史页面列表。
- 搜索过滤。
- 选中页面恢复完整上下文。
- 复制提取内容和在新标签页打开原网页。
- 历史会话上继续提问。
- 删除页面并按同步状态选择软删或硬删。

## 6. 错误与异常处理

- 页面记录损坏时只影响当前右侧，不影响其他页面浏览。
- 标题保存失败时回退旧标题。
- 删除失败时不提前清空右侧视图。

## 7. 数据与状态

- UI 运行态：
  - 当前搜索词
  - 当前选中页面
  - 列表滚动位置
  - 标题编辑态
  - 右侧页面恢复状态
- 持久化依赖：
  - `PageRecord`
  - `ConversationRecord`
  - `LoadingStateRecord`

## 8. 依赖与协作模块

- `dao/page-repository.md`
- `dao/conversation-repository.md`
- `Services/runtime-messaging.md`
- `Services/llm-dispatch.md`

## 9. 约束与禁止事项

- 左侧列表必须以页面元数据为 source of truth。
- 删除按钮不能顺带触发页面选中。
- 打开原网页只开新标签，不改变右侧当前选中页面。
- 继续对话行为要与侧边栏保持一致。

## 10. 测试要求

- 职责测试：列表、搜索、恢复、继续对话、标题编辑、删除。
- 正常流测试：恢复已有页面并继续提问、编辑消息、继续新增分支。
- 正常流测试：复制提取内容、打开原网页不改变当前选中。
- 边界测试：空状态、无搜索结果、损坏页面数据。
- 错误流测试：标题保存失败、删除失败。
- 异常流测试：有 loading 状态时恢复页面。
- 不变量测试：删除页面不影响其他页面记录。
- 可观测性测试：列表更新时间与错误状态清晰可见。
