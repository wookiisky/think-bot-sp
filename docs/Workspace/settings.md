# 设置页

## 1. 模块定位

设置页是全部长期配置的唯一编辑入口，负责模型、快捷输入、主题语言、内容提取、同步、黑名单和配置导入导出。

## 2. 核心抽象

- 基础设置
- 快捷输入
- 模型配置
- 同步配置
- 黑名单
- 导入导出

## 3. 能力边界

负责：

- 展示和编辑完整 `ExtensionConfig`。
- 对输入做结构校验。
- 触发保存、保存并同步、恢复默认、连接测试。
- 即时预览主题和语言。
- 从项目公开模板源拉取并导入快捷输入模板。

不负责：

- 直接执行模型请求。
- 直接写远端同步文件。
- 直接操作历史页面或聊天记录。

## 4. 对外接口

- 获取完整配置。
- 保存完整配置。
- 恢复默认配置。
- 测试同步连接。
- 拉取远端快捷输入模板。
- 导入远端快捷输入模板到本地配置。
- 执行导入导出。

## 5. 关键流程

- 模型新增、复制、启停、排序、删除。
- 快捷输入新增、排序、自动触发、专属分支模型配置。
- 从远端模板源导入快捷输入并重建本地稳定 ID，未知的分支模型引用在导入时过滤并提示。
- 主题与语言即时预览、保存后持久化。
- Gist/WebDAV 连接测试和保存并同步。

## 6. 错误与异常处理

- 保存失败时保留当前表单。
- 导入失败时不覆盖现有配置。
- 同步失败时保留错误信息和本地配置。
- 非法黑名单规则不允许保存。
- 远端快捷输入模板结构非法或网络失败时，不改动当前本地快捷输入列表。

## 7. 数据与状态

- UI 运行态：
  - 当前导航分栏
  - 未保存变更标记
  - 表单校验错误
  - 最近一次错误展示
  - 远端模板拉取状态
- 持久化依赖：
  - `ExtensionConfig`
  - `SyncSnapshot.lastSyncAt`

## 8. 依赖与协作模块

- `dao/config-repository.md`
- `Services/sync.md`
- `Services/i18n.md`
- `Services/icon-assets.md`

## 9. 约束与禁止事项

- 左侧栏目切换不能丢未保存表单。
- 默认模型候选只允许“启用且配置完整”的模型。
- 国际化只允许使用平铺 key。
- 图标资源不能通过在线字体加载。
- 远端快捷输入模板导入只能追加新的本地快捷输入，不允许直接覆盖现有本地项的 ID。

## 10. 测试要求

- 职责测试：模型、快捷输入、同步、黑名单、导入导出。
- 边界测试：不完整模型、重复 ID、非法规则。
- 正常流测试：保存、排序、语言切换、主题切换、远端模板导入。
- 错误流测试：保存失败、导入失败、同步失败、远端模板拉取失败。
- 异常流测试：快速重复保存、切换栏目后保持表单。
- 不变量测试：软删除配置项不破坏历史引用。
- 可观测性测试：最近同步时间和错误信息展示。
