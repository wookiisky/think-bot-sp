# 设置页核心回归

## 1. 关联能力

- 模型配置
- 快捷输入配置
- 主题语言
- 同步设置
- 黑名单
- 远端快捷输入模板导入
- 导入导出

## 2. 核心职责清单

- 正确编辑并保存完整配置。
- 只暴露可用模型候选。
- 支持快捷输入排序和自动触发配置。
- 支持从公开模板源导入快捷输入。
- 支持中英文切换。
- 支持测试同步连接与保存并同步。

## 3. 验证点

- 正常流：
  - 新增、复制、启停、删除模型。
  - 新增、排序、删除快捷输入。
  - 从远端模板源导入快捷输入并生成新的本地 ID。
  - 切换主题和语言。
  - 导出配置并导入恢复。
- 错误流：
  - 保存失败。
  - 导入结构非法。
  - 同步测试失败。
  - 远端模板源返回非法结构或网络失败。
- 异常流：
  - 未保存表单切换栏目。
  - 快速重复保存。

## 4. 关键边界条件

- 模型不完整但启用。
- 快捷输入 ID 冲突。
- 语言资源缺失 key。
- 黑名单规则非法。

## 5. 历史高风险场景

- 排序后保存结果与 UI 不一致。
- 语言切换即时预览但未持久化。
- 删除配置项导致历史引用失效。
- 远端模板导入覆盖了现有快捷输入 ID。

## 6. 测试数据与环境

- 中英文语言资源文件
- 4 类 Provider 示例模型
- 默认黑名单规则与用户自定义规则
- Gist/WebDAV 测试配置
