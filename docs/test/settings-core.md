# 设置页核心回归

## 1. 关联能力

- 模型配置
- 标签页配置
- 主题语言
- 同步设置
- 黑名单
- 远端快捷输入模板导入
- 导入导出

## 2. 核心职责清单

- 正确编辑并保存完整配置。
- 只暴露可用模型候选。
- 支持标签页排序和自动触发配置。
- 支持标签页折叠展开和名称/消息预览。
- 支持从公开模板源导入快捷输入。
- 支持中英文切换。
- 支持测试同步连接与保存并同步。
- 支持模型行摘要、展开编辑和 Provider 差异字段编辑。
- 支持本地缓存占用查看与清理。

## 3. 验证点

- 正常流：
  - 新增、复制、启停、删除模型。
  - 模型拖拽排序后保存并重新进入页面，顺序保持一致。
  - 模型行展开后编辑显示名称、Provider、模型标识和差异字段并成功保存。
  - API Key 默认掩码显示，切换查看后不影响保存值。
  - 新增、排序、删除标签页。
  - 标签页折叠展开后名称与消息预览正确。
  - 从远端模板源导入快捷输入并生成新的本地 ID。
  - 切换主题和语言。
  - 正确展示本地缓存占用并执行清理。
  - 导出配置并导入恢复。
- 错误流：
  - 保存失败。
  - 导入结构非法。
  - 同步测试失败。
  - 远端模板源返回非法结构或网络失败。
  - Provider 切换后缺少必填字段时阻止保存。
  - 缓存清理失败时错误可见且不破坏当前表单。
- 异常流：
  - 未保存表单切换栏目。
  - 快速重复保存。
  - 模型展开项切换到其他模型后再切回，未保存内容仍在当前表单态中。

## 4. 关键边界条件

- 模型不完整但启用。
- 快捷输入 ID 冲突。
- 语言资源缺失 key。
- 黑名单规则非法。
- Provider 切换后旧字段残留。

## 5. 历史高风险场景

- 排序后保存结果与 UI 不一致。
- 语言切换即时预览但未持久化。
- 删除配置项导致历史引用失效。
- 远端模板导入覆盖了现有快捷输入 ID。
- API Key 掩码逻辑导致空值误覆盖真实值。
- 缓存清理误删仍需保留的配置数据。

## 6. 测试数据与环境

- 中英文语言资源文件
- 4 类 Provider 示例模型
- 默认黑名单规则与用户自定义规则
- Gist/WebDAV 测试配置

## 7. 验收补充

- 左侧导航固定为“基础设置 / 标签页 / 语言模型 / 云同步 / 黑名单设置”。
- 顶部操作固定为“保存 / 导出配置 / 导入配置 / 重置为默认值 / 保存并同步”。
- 语言模型页采用“列表摘要 + 展开编辑”形态，不使用整页跳转编辑。
- 默认模型候选、侧边栏模型选择器、分支模型选择器对“启用且配置完整”的判定一致。
