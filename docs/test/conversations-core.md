# 对话管理页核心回归

## 1. 关联能力

- 历史页面列表
- 搜索
- 恢复会话
- 左右栏拖拽
- 继续对话
- 标题编辑
- 模型选择、图片输入、停止、清空、导出
- 打开原网页与复制提取内容
- 删除页面

## 2. 核心职责清单

- 按页面维度展示历史。
- 恢复页面标题、URL、提取内容、标签和聊天。
- 维持左右双栏工作台布局和拖拽分隔条。
- 继续对话行为与侧边栏一致。
- 支持复制提取内容和打开原网页但不打断当前上下文。
- 支持标题编辑、页面删除和删除语义分流。

## 3. 验证点

- 正常流：
  - 列表加载。
  - 搜索过滤。
  - 选中页面恢复完整上下文。
  - 左右栏拖拽后布局更新且当前上下文不丢失。
  - 继续提问。
  - 图片输入、模型切换、停止、清空、导出可用。
  - 打开原网页后右侧当前选中上下文保持不变。
  - 标题支持 `Enter` 保存、`Esc` 取消、失焦保存。
  - 编辑用户消息、继续新增分支和 Markdown 展示行为与侧边栏一致。
- 错误流：
  - 页面数据损坏。
  - 标题保存失败。
  - 删除失败。
- 异常流：
  - 页面有 loading 状态时恢复。
  - 分支 loading 与错误状态恢复。

## 4. 关键边界条件

- 空列表。
- 搜索无结果。
- 当前选中页面被删除。
- 同步开启时删除走软删。
- 同步关闭时删除走硬删。

## 5. 历史高风险场景

- 列表来源错误地依赖聊天内容反推。
- 删除按钮触发页面选中。
- 标题编辑失败后 UI 未回滚。
- 对话页输入区与侧边栏交互漂移。
- 拖拽布局导致当前标签和草稿丢失。

## 6. 测试数据与环境

- 多条历史页面记录
- 同一页面下多个标签会话
- 含 loading 的历史记录
- 含分支结果和分支错误的历史记录
- 同步开启与关闭两套删除场景
