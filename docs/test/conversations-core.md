# 对话管理页核心回归

## 1. 关联能力

- 历史页面列表
- 搜索
- 恢复会话
- 继续对话
- 标题编辑
- 打开原网页与复制提取内容
- 删除页面

## 2. 核心职责清单

- 按页面维度展示历史。
- 恢复页面内容、标签和聊天。
- 继续对话行为与侧边栏一致。
- 支持复制提取内容和打开原网页但不打断当前上下文。
- 支持标题编辑和页面删除。

## 3. 验证点

- 正常流：
  - 列表加载。
  - 搜索过滤。
  - 选中页面恢复完整上下文。
  - 继续提问。
  - 打开原网页后右侧当前选中上下文保持不变。
  - 编辑用户消息和继续新增分支行为与侧边栏一致。
- 错误流：
  - 页面数据损坏。
  - 标题保存失败。
  - 删除失败。
- 异常流：
  - 页面有 loading 状态时恢复。

## 4. 关键边界条件

- 空列表。
- 搜索无结果。
- 当前选中页面被删除。

## 5. 历史高风险场景

- 列表来源错误地依赖聊天内容反推。
- 删除按钮触发页面选中。
- 标题编辑失败后 UI 未回滚。

## 6. 测试数据与环境

- 多条历史页面记录
- 同一页面下多个标签会话
- 含 loading 的历史记录
