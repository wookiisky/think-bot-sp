# 同步与删除回归

## 1. 关联能力

- Gist 同步
- WebDAV 同步
- 配置快照推送

## 2. 核心职责清单

- 正确生成和推送快照。
- 正确测试连接。
- 最近同步时间展示正确。
- 页面删除时同步开启会先写入 tombstone，再清理本地页面、会话与 loading。
- 当前阶段仍未完成远端拉取合并与删除传播闭环。

## 3. 验证点

- 正常流：
  - Gist 连接测试成功。
  - WebDAV 连接测试成功。
  - 手动同步更新最近同步时间。
  - 同步反馈展示快照大小。
- 错误流：
  - 鉴权失败。
  - 网络失败。
- 异常流：
  - service worker 启动后再注入测试 provider。
  - 同步失败后再次重试。

## 4. 关键边界条件

- 空数据集。
- 缺少 Gist Token / Gist ID。
- 缺少 WebDAV URL。

## 5. 历史高风险场景

- service worker 重启或惰性回收后，测试桩注入失效导致误走真实网络。
- 同步失败覆盖本地有效数据。
- 最近同步时间显示与真实状态不一致。

## 6. 测试数据与环境

- 启用同步的配置
- 远端错误桩
