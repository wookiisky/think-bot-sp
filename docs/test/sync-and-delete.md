# 同步与删除回归

## 1. 关联能力

- Gist 同步
- WebDAV 同步
- 软删除
- 硬删除
- 冲突合并
- 快照导入导出

## 2. 核心职责清单

- 正确生成和推送快照。
- 同步开启时删除写墓碑。
- 同步关闭时删除做物理清理。
- 本地与远端并发修改时按对象粒度正确合并。
- 最近同步时间展示正确。

## 3. 验证点

- 正常流：
  - Gist 连接测试成功。
  - WebDAV 连接测试成功。
  - 保存并同步更新最近同步时间。
  - 本地改配置、远端改历史页面后同步结果同时保留两侧最新数据。
- 错误流：
  - 鉴权失败。
  - 网络失败。
  - 远端格式非法。
- 异常流：
  - 同步进行中删除页面。
  - 同步失败后再次重试。

## 4. 关键边界条件

- 空数据集。
- 大量页面和会话。
- 已存在墓碑的重复删除。

## 5. 历史高风险场景

- 软删除过滤失效导致已删页面复活。
- 同步失败覆盖本地有效数据。
- 最近同步时间显示与真实状态不一致。
- 整份快照覆盖导致较新的局部对象被回退。

## 6. 测试数据与环境

- 启用同步的配置
- 至少一条历史页面与多标签会话
- 远端错误桩
