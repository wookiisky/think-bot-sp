# 语言资源仓储

## 1. 关联模块

- 国际化服务
- 设置页
- 侧边栏
- 对话管理页

## 2. 访问数据域

- `DataSchema/locale-resource.md`

## 3. 提供的能力

- 加载 `zh-CN` 与 `en` 静态资源。
- 提供 typed key 索引。
- 提供按 key 查询和回退。
- 校验两种语言资源 key 是否一致。

## 4. 事务边界与并发约束

- 语言资源是构建期静态产物，不做运行时写入。
- 当前语言切换只影响活跃页面状态，不影响资源本体。

## 5. 上层依赖边界

- 上层只依赖 `t(key)` 和语言切换接口。
- 上层不直接读取 YAML 文件。

## 6. 需要验证的点

- 语言文件解析正确。
- 缺失 key 回退正确。
- typed key 与资源文件一致。
