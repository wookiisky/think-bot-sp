# 配置仓储

## 1. 关联模块

- 设置页
- 侧边栏
- 对话管理页
- 模型调度
- 同步服务

## 2. 访问数据域

- `DataSchema/config.md`

## 3. 提供的能力

- 读取完整配置。
- 保存完整配置。
- 恢复默认配置。
- 校验并导入配置。
- 把远端快捷输入模板规范化为本地 `QuickInput` 并并入配置。
- 提供衍生查询：
  - 默认模型
  - 启用且配置完整的模型
  - autoTrigger 快捷输入
  - 当前主题与语言
  - 快捷输入名称与消息预览摘要

## 4. 事务边界与并发约束

- 配置按单对象原子写入。
- 保存前必须通过 schema 校验。
- 每次成功保存、导入或模板并入都必须刷新 `ExtensionConfig.updatedAt`。
- 不提供页面直接写入局部 key 的能力，避免并发覆盖。

## 5. 上层依赖边界

- 上层可以依赖 typed query 和 save 接口。
- 上层不能依赖原始 key 名和存储布局。
- 上层不能绕过仓储直接读写 `chrome.storage.local`。

## 6. 需要验证的点

- 保存后跨页面读取一致。
- 默认模型只返回启用且完整的模型。
- 导入失败不污染现有配置。
- 快捷输入和模型软删除后历史引用仍稳定。
- 远端模板导入后本地 ID 稳定且不覆盖现有项。
- 远端模板中的未知模型引用在导入时被过滤且不会阻断整体导入。
- 快捷输入预览摘要与折叠列表所需字段稳定可读。
